# The application is not currently responsive (Will likely cause visual/functional issues on vertical screens, smaller screens, mobile devices)

Scrolling items represent the current top priced items for sale and their change within the last 48 hours:

  • Red Text applies to negative changes in price in the last 48 hours
  • Green Text applies to positive changes in price in the last 48 hours

If you scroll down:

  • Server statuses are shown in the bottom left under "Server Statuses":
    • A green border represents no status issues with that current service
    • A red border represents a current status issue with that current service
  • "PMC Counts" shows all available maps and the number of PMCs that can spawn into a raid on the given map

The home screen consists of one action item (search bar):

  • Enter a search term into this box and press enter
  • You will be given a popup modal including the results
  • At any time, you can exit this modal and start over by simply clicking outside of the modal
  • This modal will be scrollable if the results include multiple items
  • If no items are shown in the modal (Click outside of the modal to close it and perform another search)
  • Once you have located your item within the results, simply clicking (or hovering and pressing "Enter") will open the final modal containing specific information for the chosen item:
    • The name of the selected item is at the top
    • "Last Updated:" is the time all of the shown information was updated within Tarkov's API ("How recent are the numbers")
      • If the text is green (item has been updated within the last 30 minutes)
      • If the text is yellow (item has been updated within 30 min-1hr)
      • If the text is red (item information is more than an hour old) (items that can't be sold on Flea aren't updated as often)
    • "Avg 24hr Price:" average listing price on the Flea for the last 24 hours
    • "Last 48hr Change:" change in listing prices on the Flea in the last 48 hours
      • If the text is green (prices have risen in the last 48 hours)
      • If the text is red (prices have dropped in the last 48 hours)

"Need For Tasks":

  • If color of the text is green, the item is needed for a task
  • If multiple tasks are listed, the container will be scrollable for viewing
  • The Task card will show Trader/Task Name/Minimum Required Level for the tasks (Might be able to track which quests the user has already completed in the future)

Next will be an image of the item that you are inspecting

Sell Prices:

List all prices that a trader will pay for the item and the cost of that item on the Flea at the time the card was updated ("Last Updated:")

  • If the text is green and bold (That is the best option for selling the item) (AT UPDATED TIME)

Buy Prices:

List all places you can purchase the item and its cost (Flea price will be the cost of that item on the Flea at the time the card was updated ("Last Updated:"))

  • If the text is green (That is the best option for purchasing the item) (AT UPDATED TIME)

If the item can be purchased from anywhere and sold back to a trader for no more than a RUB 500 loss, the card will have a "QUICK MONEY MAKER" animation above the image. (These items will likely be able to be purchased on the Flea and sold to traders for a profit)
